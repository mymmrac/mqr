## :gear: Install

```shell
go install github.com/mymmrac/mqr@latest
```

## :zap: Usage

As CLI

```shell
mqr "Hello Gopher!"
```

> Will print QR code containing data `Hello Gopher!`

As TUI

```shell
mqr
```

> Will launch TUI where you can enter (or paste) your data
